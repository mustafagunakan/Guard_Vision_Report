Guard Vision: Smarter AI for Theft Detection

Project Overview

1. Overview

Title: Guard Vision: Redefining Front-Door AI Surveillance

Objective: Develop an AI model capable of accurately detecting potential theft incidents in gardens and front doors, reducing irrelevant notifications and enhancing user trust.

Team Composition:

  • Team Lead
  • Scrum Master
  • Product Designer
  • Data Scientists (5-7 members)

2. Research and Model Assessment

The team evaluated several AI detection models based on visual data to identify the most suitable option.

  • Model A: High false positives under low-light conditions.
  • Model D: Best accuracy-speed balance.
  • Model E: Low precision in real-time detection testing.

3. Model Selection and Integration

Chosen Model: Model D

Challenges:

  • Object ID Loss: Implemented re-identification mechanism.
  • Lag: Transitioned to YOLOv11 small model for balance.

4. Technologies Used

  • AI Models: YOLOv11 small/large
  • Programming: TensorFlow, PyTorch
  • Tools: Jupyter Notebook, Docker, GitHub

5. Next Steps

  • Refinement of tracking algorithms.
  • Testing with diverse datasets.
  • Exploration of commercialization opportunities.
